在命令行输出Hello World

在Windows下，按 win+R ,调出运行窗口，输入cmd，打开命令提示符。

在命令行中输入命令 python，进入Python交互模式。

输入python语句 print('Hello World') ，会看到输出了一串字符：Hello World：

这样，你就成功运行了第一个python程序！
